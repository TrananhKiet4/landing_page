# SmartAir Pro Landing Page

## Giới thiệu

SmartAir Pro là landing page giới thiệu máy lọc không khí thông minh tích hợp AI. Website được xây dựng theo yêu cầu bài test vòng 2 vị trí Thực tập sinh IT Phát triển Website tại HELICORP.
